```python
sidebar(defaultopen = True) -> Dict
```

The `sidebar` function adds sidebar to your app.

## Parameters

- **`defaultopen`** _(True)_: Whether to expand sidebar on desktop by default

## Returns

- `Dict` with sidebar component data

## Usage Example

Here's an example of how to add sidebar to your app:

```python
from preswald import sidebar

sidebar(defaultopen=True)
```
